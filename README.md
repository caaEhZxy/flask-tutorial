# flask-tutorial

此仓库用于学习 Flask+Gunicorn+Docker+Nginx+Jenkins 自动构建、上传镜像和部署项目。
```
flask-tutorial  
├── Dockerfile   
├── Jenkinsfile                 
├── Pipfile  
├── Pipfile.lock  
├── README.md  
├── docker-compose.yaml  
├── flaskr  
│   ├── __init__.py  
│   └── templates  
│       └── index.html  
├── gunicorn.sh  
├── nginx.conf  
└── requirements.txt
```
